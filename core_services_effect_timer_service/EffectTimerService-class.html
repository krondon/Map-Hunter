EffectTimerService

EffectTimerService class

Service responsible for managing active effect timers.

Extracted from PowerEffectProvider to follow Single Responsibility Principle. This class handles ONLY:

  • Storing active effects
  • Managing Timer objects
  • Emitting events when effects change

Security Note: Local expiration times are validated against DB values to prevent clients from shortening effect durations.
