# Audio Assets

## Story Background Music

Para la pantalla de introducción de la historia "Protocolo Asthoria", se recomienda usar:

**Archivo requerido:** `story_intro.mp3`

### Características recomendadas:
- Estilo: Música épica/cinematográfica con elementos cyberpunk
- Duración: 2-3 minutos (con loop)
- Mood: Misterioso, épico, futurista
- Tempo: Moderado (80-100 BPM)
- Instrumentación: Sintetizadores, cuerdas orquestales, percusión épica

### Fuentes sugeridas de música libre de derechos:
- Pixabay Music: https://pixabay.com/music/
- Free Music Archive: https://freemusicarchive.org/
- YouTube Audio Library
- Incompetech: https://incompetech.com/

### Ejemplo de búsqueda:
- "Epic Cinematic Music"
- "Cyberpunk Ambient"
- "Sci-Fi Adventure Music"
- "Time Travel Theme"

Una vez descargada la música, colócala en esta carpeta con el nombre `story_intro.mp3`
